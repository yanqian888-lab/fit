## 九、下一步建议

1. 用 HBuilderX 打开 `frontend` 目录，运行到内置浏览器或 H5 预览。
2. 准备一台 Android 真机，开启「开发者选项」和「USB 调试」，连接电脑后运行 `adb devices` 验证。
3. 在 HBuilderX 中使用「运行 → 运行到手机或模拟器 → 制作自定义基座」进行真机调试。
4. 后端服务已运行，前端 `request.js` 中的 `BASE_URL` 当前为 `http://localhost:3000/api`，真机调试时请改为电脑局域网 IP（如 `http://192.168.x.x:3000/api`）。
